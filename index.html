視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓背景比前景移動得更慢。 當訪客向下捲動時會產生 3D 的立體感，為畫面增添深度並提升沉浸式的瀏覽體驗。 視差效果基於視覺錯覺：人眼會把較近的物體看得比遠處更大，因此會覺得遠處的物體移動得更慢。 這種錯覺早已被運用在不同媒介中，帶來更寫實的效果；最早可追溯到迪士尼《白雪公主》這類傳統動畫， 以及《超級瑪利歐》等電玩遊戲。 隨著 CSS 和 HTML 的進步，視差效果也逐漸演變成今日我們熟悉的網頁設計手法。

首批在瀏覽器實作視差滾動的範例由網頁開發者 Glutnix 在 2007 年的部落格文章中分享， 提供了使用 JavaScript 與 CSS 2 的範例程式碼，支援當時的 IE6 等瀏覽器。 直到 2011 年 HTML5 與 CSS3 推出後，這項技巧才在設計師間普及。 擁護者認為這是展現網頁流動性的簡單方法，也能透過視差背景提升用戶參與度並改善整體體驗。 不過，普渡大學在 2013 年的研究指出：「…雖然視差滾動提升了某些使用體驗面向，但未必能改善整體體驗。」


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要原因：

  • 有助於說故事

    視差圖像非常適合用來說故事。 優秀的設計重在溝通，而設計網站時，我們希望與訪客交流，講述產品、服務或品牌的故事。 視差滾動能讓訪客沉浸在獨特的線上體驗中；甚至可以用單頁網站就帶完一段完整的品牌故事， 不需要跳轉到其他頁面。

  • 有助提升用戶互動

    視差圖像也能提升用戶互動。降低網站跳出率是設計時的重要目標之一； 跳出率表示僅瀏覽單一頁面後就按返回或關閉分頁的訪客比例。 高跳出率會影響搜尋排名，因此任何能降低跳出率的作法都很值得嘗試。 精心設計的視差頁面能帶來動態體驗，讓使用者願意停留更久。


如果想在網站加入視差效果，可以考慮以下建議：

  • 選擇性使用

    視差效果應用在特定元素時更有力量，而不是所有頁面都使用。 你可以在頁首、標題或僅首頁採用，讓視差加強網站設計，而非分散注意力。

  • 圖像壓縮

    視差滾動通常需要多種媒體與 CSS 形狀來營造深度。 不要因為過大的圖片檔案拖慢載入速度，影響效能與體驗。

  • 運用色彩

    視差不僅與前景、背景的圖片有關，色彩也同樣重要。 你可以在背景使用較低飽和的色調，前景採用更鮮明的顏色，強化深度感。
